# Avik Portfolio — Frontend Only

This is a frontend-only Next.js (app router) + TypeScript + Tailwind CSS + lucide-react portfolio.
No backend, no database, no email — ready to run and deploy.

## Quick start

1. Install dependencies:
   ```
   npm install
   ```
2. Start dev server:
   ```
   npm run dev
   ```
3. Open http://localhost:3000

## Deploying

- Deploy to Vercel by connecting this repository.
